# Fullstack Chat App

A simple fullstack chat application built with **React** on the frontend and **Node.js / Express** on the backend. This project demonstrates real-time chat functionality, API integration, and frontend/backend state management.

---

## Features

- Responsive chat interface built with **React** and **Tailwind CSS**
- Real-time chat between user and backend
- Frontend state management using **React Hooks**
- Async POST requests with **Fetch API**
- Modular component structure (`ChatInput`, `App`)
- Easily extendable for AI or database integration

---

## Tech Stack

- **Frontend:** React, Tailwind CSS, JavaScript
- **Backend:** Node.js, Express.js, CORS
- **Communication:** REST API (POST `/chat`)
- **Environment Management:** dotenv for backend API keys / config
- **Optional Future:** OpenAI API, database (MongoDB / PostgreSQL)
